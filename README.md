# 微前端技术分享API服务
基于NestJs + TypeScript + TypeORM + Redis + MongoDB 
> 数据截取于中医数仓部分病例数据

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
